# LIFT GitHub Pages Upload

## Upload steps
1. Create a new public GitHub repository.
2. Upload all files from this folder to the root of the repository.
3. In GitHub: Settings -> Pages.
4. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Save.
6. Wait about a minute. Your site URL will appear on the Pages screen.
7. Open that URL in Safari on your iPhone.
8. Tap Share -> Add to Home Screen -> Add.
